This directory contains the source for the IBM Research Jikes Compiler.

File Makefile contains a makefile with a number of clauses at the start
giving the options needed for several systems. All the clauses have been
commented out. You need to uncomment the clause for your system before
using this file.

You may also be able to build the system by just typing
	./configure
This build mode was added on 9 March 1999, and is still undergoing testing.
